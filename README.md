# Certified-Information-Security-Engineer-CISE-
Certified Information Security Engineer (CISE) - 정보보안기사 공부자료 정리 봇

## 소개

정보보안기사 시험 준비를 위한 CLI 기반 공부자료 관리 도구입니다.
과목별로 공부 노트를 체계적으로 관리하고, 퀴즈 기능으로 복습할 수 있습니다.

## 주요 기능

- **공부 노트 관리**: 추가, 조회, 수정, 삭제, 키워드 검색
- **과목별 분류**: 정보보안기사 5개 과목 및 세부 주제별 정리
- **퀴즈 기능**: 객관식/주관식 퀴즈 출제 및 풀기
- **학습 통계**: 과목별 학습 진도, 퀴즈 정답률, 복습 횟수 추적
- **중요도 관리**: 1~5단계 중요도 설정
- **학습 완료 표시**: 노트별 학습 완료 상태 관리

## 시험 과목 구조

| 번호 | 과목명 | 세부 주제 |
|------|--------|-----------|
| 1 | 시스템 보안 | 운영체제, 서버, 리눅스/유닉스, 윈도우, 악성코드 |
| 2 | 네트워크 보안 | 네트워크 기초, 공격/방어, 방화벽, IDS/IPS, VPN |
| 3 | 어플리케이션 보안 | 웹 보안, DB 보안, 전자상거래, 소프트웨어 개발 보안 |
| 4 | 정보보안 일반 | 암호학, 접근통제, 인증, 보안 모델, PKI |
| 5 | 정보보안 관리 및 법규 | ISMS, 개인정보보호법, 정보통신망법, 위험관리 |

## 사용법

### 실행

```bash
cd study_bot
python main.py
```

### 샘플 데이터 로드 (선택)

처음 사용 시 예시 노트와 퀴즈를 추가할 수 있습니다:

```bash
cd study_bot
python sample_data.py
```

### 메뉴 구조

```
[1]  공부 노트 추가       - 새 노트를 과목별로 작성
[2]  노트 목록 보기       - 전체 노트를 과목별로 조회
[3]  과목별 노트 보기     - 특정 과목의 노트만 조회
[4]  노트 검색            - 키워드로 노트 검색
[5]  노트 상세 보기       - 특정 노트 내용 확인 (복습 카운트 증가)
[6]  노트 수정            - 기존 노트 내용 수정
[7]  노트 삭제            - 노트 삭제
[8]  퀴즈 풀기            - 등록된 퀴즈를 랜덤으로 풀기
[9]  퀴즈 추가            - 새 퀴즈 문제 등록
[10] 학습 통계            - 과목별 학습 현황 및 퀴즈 정답률
[11] 과목 구조 보기       - 시험 과목 및 세부 주제 확인
[0]  종료
```

## 프로젝트 구조

```
study_bot/
├── main.py          # 프로그램 진입점
├── bot.py           # 메인 봇 로직 및 메뉴 처리
├── models.py        # 데이터 모델 (StudyNote, QuizQuestion)
├── storage.py       # JSON 파일 기반 데이터 저장소
├── quiz.py          # 퀴즈 기능
├── sample_data.py   # 샘플 데이터 로더
└── data/            # 데이터 저장 디렉토리
    ├── notes.json   # 노트 데이터
    └── quiz.json    # 퀴즈 데이터
```

## 요구사항

- Python 3.7 이상
- 외부 라이브러리 불필요 (표준 라이브러리만 사용)
